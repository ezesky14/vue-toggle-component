# Vue Toggle component

This is a project for transition element which height is auto.

## Installation

#### NPM
```bash
npm install --save vue-toggle-component@latest
```

## Import

```javascript
import VueToggle from 'vue-toggle-component'
```

## Example
```
<vue-toggle title="Toggle me" :id="'1'" />
```
